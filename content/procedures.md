## I. 공통 감리절차

단계 별 감리는 예비조사 → 현장감리 → 시정조치 확인의 순서로 이루어지며, 각 단계 별 보고서 제출

```mermaid
    style A fill:#e1f5fe
    style B fill:#e8f5e8
    style C fill:#f3e5f5
```

## II. A00. 예비조사

가. 예비조사 절차도

```mermaid
graph TD
    A([  예비조사 시작  ]) --> B([  A10 예비조사 준비  ])
    B --> C([  A20 예비조사 실시  ])
    C --> D([  A30 감리계획서작성  ])

    style A fill:#e1f5fe
    style B fill:#e8f5e8
    style C fill:#e8f5e8
    style D fill:#f3e5f5
```

나. 세부 절차

| 세부 절차 | 설명 |
|----------|------|
| A10. 예비조사 준비 | – 예비조사/현장감리 실시가능여부확인<br>– 실시 일정, 투입감리원 확정 등 준비 |
| A20. 예비조사 실시 | – 사업관리문서 및 산출물 접수/검토<br>– 감리계획서 작성에 필요사항 도출 |
| A30. 감리계획서 작성 및 제출 | – 예비조사 실시 결과 반영하여 감리 범위, 절차, 수행방법 작성/협의/확정<br>최종 결과물로 감리계획서가 도출되며 발주자 및 사업자에게 공문과 함께 제출 |

## III. B00. 현장감리

가. 현장감리 절차도

```mermaid
graph TD

    D --> E([  B40 보고서안작성  ])

    style A fill:#e1f5fe
    style B fill:#e8f5e8

    style G fill:#f3e5f5
```

나. 현장감리 세부 절차

| 세부 절차 | 설명 |
|----------|------|
| B10. 감리 시작 | – 현장감리 수행에 필요한 환경 준비<br>– 착수 회의 시간 및 장소 확인/준비 |
| B20. 착수 회의 | – 감리법인, 발주자, 사업자 모두 참석<br>– 감리수행 관련 세부 절차/내용 공유 |
| B30. 감리 수행 | – 산출물 접수, 검토, 분석, 시험 수행<br>– 문제점 발견 및 개선방향 도출, 면담 |
| B40. 보고서(안) 작성 및 검토 | – 감리수행결과보고서 초안 작성/검토<br>– 문제점 및 개선 필요사항 항목 중심 |
| B50. 종료 회의 | – 감리수행결과보고서(안) 기반 회의<br>– 보고서 설명 수준, 방법 등 논의 |
| B60. 보고서 확정 및 통보 | – 종료회의과정 도출된 이견 확인/반영<br>– 공문으로 발주자 및 사업자에게 통보<br>최종적으로 감리수행결과보고서가 도출되며, 공문과 함께 발주자 및 사업자에게 제출 |

```mermaid
    C --> D([  C30 시정조치확인서작성  ])

```

나. 시정조치 확인 세부 절차

| 세부 절차 | 설명 |
|----------|------|
| C10. 확인 준비 | – 개선권고사항 시정 결과에 따라 시정조치 확인 일정, 확인 방법 등 확정 |
| C20. 시정조치 확인 | – 시정조치확인계획에 따라 현장에서 결과확인, 면담실시, 미조치 사유파악 |
| C30. 시정조치확인보고서 작성/협의 | – 시정조치결과 확인 후 발견한 미/오조치내용 사실확인, 보고서작성/제출 |
| C40. 시정조치확인보고서 확정/제출 | – 시정조치확인보고서 초안에 대해 발주자, 사업자 확인 후 공식 제출<br>최종적으로 시정조치확인보고서가 도출되며, 발주자 및 사업자에게 공문과 함께 제출 |
